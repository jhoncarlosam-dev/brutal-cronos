# BRUTAL CRONOS

Temporizador y cronómetro con diseño brutalista.

## Características

- **Modo Temporizador**: Cuenta regresiva configurable con presets
- **Modo Cronómetro**: Cronómetro con registro de vueltas
- **Diseño Responsive**: Adaptado a todas las pantallas
- **Atajos de Teclado**:
  - **Espacio**: Iniciar/Pausar
  - **R**: Reiniciar
  - **L**: Marcar vuelta (solo en modo cronómetro)

## Uso

1. Abre el archivo `index.html` en tu navegador web
2. Selecciona el modo (Timer o Cronómetro)
3. ¡Empieza a usar!

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
